Do código ao indicador: automatizando o PCM com Python

jan/2026 - fev/2026
Projeto acadêmico - Universidade Tecnológica Federal do Paraná

Gerador de KPIs de manutenção em Python, simulando 90 dias de operação industrial. Transforma lógica de programação em indicadores que apoiam decisões no PCM (Planejamento e Controle de Manutenção).

15 equipamentos simulados MTBF, MTTR, Disponibilidade Análise de criticidade Exportação Excel/CSV
Python Pandas NumPy Análise de Dados
Arquivos gerados (7 CSVs + Excel)
  • equipamentos.csv - Dados dos equipamentos
  • ordens_manutencao.csv - Histórico de OM
  • kpi_equipamentos.csv - KPIs por equipamento
  • kpi_manutencao.csv - KPIs por tipo
  • kpi_disponibilidade.csv - Disponibilidade mensal
  • kpi_custos.csv - Análise financeira
  • kpi_tendencia.csv - Tendências
  • kpis_manutencao_completo.xlsx - Relatório consolidado
